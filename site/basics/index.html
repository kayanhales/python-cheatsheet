    Python uses indentation to define blocks (no {} needed).

    if True:
    print("Indented block")

    üí° 2. Variables

    No need to declare types ‚Äî Python is dynamically typed.

    x = 10

pi = 3.14
is_active = True

    Reassigning changes type dynamically:

    x = "Now I'm a string!"

    üîÅ 6. Conditionals

    print("Adult")

    print("Child")

    Ternary Expression:

    üîÑ 7. Loops

    For Loop

    for i in range(5):
    print(i)

    While Loop

    count = 0
while count < 5:
    print(count)
    count += 1

    Loop Control Statements

    for n in range(10):
    if n == 3:
        continue  # Skip 3
    if n == 8:
        break     # Stop at 8
    print(n)

    üß± 8. Data Structures

    Lists

    Tuples

    Sets

    Dictionaries

    üßÆ 9. List Comprehensions
