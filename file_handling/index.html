    üóÇÔ∏è Python File Handling Cheatsheet

    Master the essentials of working with files in Python ‚Äî reading, writing, managing, and handling exceptions.

    üìñ 1. Opening Files

    Use the built-in open() function.

    file = open("example.txt", "r")  # Modes: r, w, a, x, b, t
content = file.read()
file.close()

    Common Modes

    Mode Description
    r Read (default)
    w Write (overwrites file)
    a Append
    x Create (fails if file exists)
    b Binary mode
    t Text mode (default)
    r+ Read and write

    üß† 2. Using with (Context Manager)

    Automatically handles closing files ‚Äî the best practice.

    with open("data.txt", "r") as file:
    data = file.read()
print("File closed:", file.closed)

    ‚úçÔ∏è 3. Writing to Files

    with open("notes.txt", "w") as f:
    f.write("First line\n")
    f.write("Second line\n")

    üí° Using w overwrites the file; use a to append.

    üìú 4. Reading Files

    with open("notes.txt", "r") as f:
    print(f.read())        # Entire file
    f.seek(0)
    print(f.readline())    # One line
    f.seek(0)
    print(f.readlines())   # List of lines

    üß© 5. Working with Binary Files

    with open("image.png", "rb") as f:
    bytes_data = f.read()

    To write binary data:

    with open("copy.png", "wb") as f:
    f.write(bytes_data)

    üîÑ 6. Checking File Existence

    import os

if os.path.exists("notes.txt"):
    print("File exists")
else:
    print("File not found")

    Or use pathlib (modern and preferred):

    from pathlib import Path

path = Path("notes.txt")
if path.exists():
    print("File exists")

    üóëÔ∏è 7. Deleting and Renaming Files

    import os

os.rename("old_name.txt", "new_name.txt")
os.remove("unnecessary.txt")

    Or with pathlib:

    path = Path("old.txt")
path.rename("new.txt")
path.unlink()  # Deletes the file

    üìÇ 8. Directory Handling

    os.mkdir("my_folder")
os.rmdir("my_folder")
os.listdir(".")  # List files in current directory

    pathlib version:

    p = Path("my_folder")
p.mkdir(exist_ok=True)
for file in p.iterdir():
    print(file)

    ‚ö†Ô∏è 9. Handling File Exceptions

    try:
    with open("data.txt") as f:
        print(f.read())
except FileNotFoundError:
    print("File does not exist!")
except PermissionError:
    print("Permission denied!")

    üß∞ 10. Reading/Writing CSV and JSON

    CSV

    import csv

# Writing
with open("data.csv", "w", newline="") as f:
    writer = csv.writer(f)
    writer.writerow(["Name", "Age"])
    writer.writerow(["Alice", 30])

# Reading
with open("data.csv", "r") as f:
    reader = csv.reader(f)
    for row in reader:
        print(row)

    JSON

    import json

data = {"name": "Bob", "age": 25}

with open("data.json", "w") as f:
    json.dump(data, f)

with open("data.json", "r") as f:
    loaded = json.load(f)
    print(loaded)

    üßº 11. Tips & Best Practices

    Always use with open() for safety
    Use pathlib over os when possible
    Validate file paths before reading/writing
    Close files explicitly if not using with
    √üHandle exceptions for robust code
